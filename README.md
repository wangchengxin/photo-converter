# 证件照处理工具
## 介绍
通过electron创建的证件照处理工具，支持JPG、PNG、BMP等主流格式，并且可以选择主流证件照尺寸，也可以自定义尺寸。
## 运行
```
npm install
npm run start
```
## 注意事项
如果您一直卡在 **node install.json**，请删除**node_modules**文件夹执行以下命令后，重新运行
```
echo electron_mirror="https://npm.taobao.org/mirrors/electron/" >> ~/.npmrc
```
